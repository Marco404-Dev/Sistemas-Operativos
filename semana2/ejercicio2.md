# Banco de preguntas – Semana 2: Sistemas Operativos y gestión de procesos

## Preguntas de opción múltiple

### 1. ¿Qué es un sistema de cómputo?
A) Solo el hardware de una computadora  
B) Solo los programas instalados  
C) El conjunto de hardware y software que permite procesar información  
D) Únicamente la memoria RAM  

**Respuesta:** C  
**Solución:** Un sistema de cómputo está formado por hardware y software que trabajan juntos para procesar información.

---

### 2. ¿Cuál de los siguientes es un componente principal de un sistema de cómputo?
A) CPU  
B) Carpeta de tareas  
C) Navegador web  
D) Documento de Word  

**Respuesta:** A  
**Solución:** La CPU es uno de los componentes principales del sistema de cómputo.

---

### 3. ¿Cuál es el flujo básico de un sistema de cómputo?
A) Procesamiento → entrada → almacenamiento → salida  
B) Entrada → procesamiento → salida → almacenamiento  
C) Salida → entrada → procesamiento → almacenamiento  
D) Almacenamiento → salida → entrada → procesamiento  

**Respuesta:** B  
**Solución:** Ese es el flujo básico indicado en el tema.

---

### 4. ¿Qué es el sistema operativo?
A) Un dispositivo de entrada  
B) Un programa de oficina  
C) El software que administra los recursos del sistema  
D) Un tipo de memoria  

**Respuesta:** C  
**Solución:** El sistema operativo administra los recursos y organiza el funcionamiento de la computadora.

---

### 5. ¿Cuál NO es una función principal del sistema operativo?
A) Gestionar memoria  
B) Administrar dispositivos  
C) Ejecutar programas  
D) Fabricar hardware  

**Respuesta:** D  
**Solución:** El sistema operativo administra el hardware, pero no lo fabrica.

---

### 6. El proceso de arranque de la computadora también se llama:
A) DMA  
B) Bootstrap  
C) Interrupción  
D) Planificación  

**Respuesta:** B  
**Solución:** Bootstrap es el nombre del proceso mediante el cual la computadora inicia el sistema operativo.

---

### 7. ¿Qué ocurre primero en el proceso de arranque?
A) El sistema operativo toma el control  
B) Se localiza el bootloader  
C) Encendido del equipo  
D) Se carga el sistema operativo en memoria  

**Respuesta:** C  
**Solución:** El primer paso es encender el equipo.

---

### 8. ¿Qué se ejecuta al inicio del arranque?
A) Word  
B) BIOS o UEFI  
C) Chrome  
D) El disco duro  

**Respuesta:** B  
**Solución:** En el arranque se ejecuta primero BIOS o UEFI.

---

### 9. ¿Qué función cumple el bootloader?
A) Apagar la computadora  
B) Localizar y ayudar a cargar el sistema operativo  
C) Guardar archivos en carpetas  
D) Aumentar la memoria RAM  

**Respuesta:** B  
**Solución:** El bootloader participa en la localización y carga del sistema operativo.

---

### 10. ¿Qué es una interrupción?
A) Una carpeta especial del sistema  
B) Una señal que envía un dispositivo al procesador para pedir atención  
C) Un error permanente del disco  
D) Un tipo de memoria secundaria  

**Respuesta:** B  
**Solución:** Una interrupción avisa al CPU que debe atender algo importante.

---

### 11. Cuando presionas una tecla, el teclado:
A) Apaga la CPU  
B) Envía una interrupción  
C) Libera la RAM  
D) Cierra el sistema operativo  

**Respuesta:** B  
**Solución:** El teclado genera una interrupción para que el procesador atienda la acción.

---

### 12. ¿Por qué las interrupciones son importantes?
A) Porque reemplazan la memoria  
B) Porque evitan que la CPU pregunte constantemente a los dispositivos  
C) Porque eliminan procesos automáticamente  
D) Porque aumentan el tamaño del disco  

**Respuesta:** B  
**Solución:** Las interrupciones hacen más eficiente el sistema, porque el dispositivo avisa solo cuando necesita atención.

---

### 13. ¿Qué significa DMA?
A) Direct Memory Access  
B) Dynamic Machine Access  
C) Digital Memory Administration  
D) Data Machine Allocation  

**Respuesta:** A  
**Solución:** DMA significa Direct Memory Access.

---

### 14. ¿Qué ventaja principal ofrece DMA?
A) Elimina la necesidad de memoria RAM  
B) Permite transferir datos sin usar constantemente la CPU  
C) Hace innecesario el sistema operativo  
D) Convierte disco en caché  

**Respuesta:** B  
**Solución:** DMA reduce la carga de la CPU y mejora la transferencia de datos.

---

### 15. En el funcionamiento de DMA, ¿quién toma el control de la transferencia?
A) El monitor  
B) El controlador DMA  
C) El teclado  
D) La impresora  

**Respuesta:** B  
**Solución:** La CPU inicia el proceso, pero luego el controlador DMA se encarga de la transferencia.

---

### 16. ¿Cómo se organiza el almacenamiento en la computadora?
A) Alfabéticamente  
B) Jerárquicamente  
C) Aleatoriamente  
D) Por colores  

**Respuesta:** B  
**Solución:** El almacenamiento se organiza en niveles jerárquicos según velocidad y capacidad.

---

### 17. ¿Cuál de estos niveles de almacenamiento es más rápido?
A) Disco HDD  
B) RAM  
C) Registros CPU  
D) Cintas magnéticas  

**Respuesta:** C  
**Solución:** Los registros CPU son los más rápidos.

---

### 18. ¿Qué es la memoria principal?
A) El disco duro  
B) La RAM  
C) La caché  
D) El USB  

**Respuesta:** B  
**Solución:** La memoria principal es la RAM, donde se cargan programas y el sistema operativo para ejecutarse.

---

### 19. ¿Qué significa que la RAM sea volátil?
A) Que nunca se llena  
B) Que sus datos se pierden al apagar la computadora  
C) Que solo guarda imágenes  
D) Que funciona sin CPU  

**Respuesta:** B  
**Solución:** La RAM pierde su contenido cuando se apaga la máquina.

---

### 20. ¿Qué almacenan los discos magnéticos?
A) Información de forma permanente  
B) Solo procesos activos  
C) Señales de audio en RAM  
D) Interrupciones del teclado  

**Respuesta:** A  
**Solución:** Los discos magnéticos, como los HDD, guardan datos de manera permanente.

---

### 21. ¿Qué es un proceso?
A) Un archivo guardado  
B) Un programa en ejecución  
C) Un periférico  
D) Una carpeta del sistema  

**Respuesta:** B  
**Solución:** Un proceso es un programa que se está ejecutando en ese momento.

---

### 22. ¿Qué hace el sistema operativo con los procesos?
A) Solo los guarda en carpetas  
B) Los crea, planifica y termina  
C) Los convierte en hardware  
D) Los mueve al disco automáticamente  

**Respuesta:** B  
**Solución:** El sistema operativo administra el ciclo de vida de los procesos.

---

### 23. ¿Qué hace el sistema operativo en la administración de memoria?
A) Decide qué programa usa memoria, cuánto usa y cuándo liberarla  
B) Solo muestra archivos en pantalla  
C) Fabrica más memoria RAM  
D) Desactiva el procesador  

**Respuesta:** A  
**Solución:** El sistema operativo reparte y controla el uso de la memoria.

---

### 24. Si la RAM no es suficiente, el sistema operativo puede usar:
A) BIOS  
B) Memoria virtual  
C) Interrupciones  
D) Bootloader  

**Respuesta:** B  
**Solución:** La memoria virtual se usa como apoyo cuando la RAM es insuficiente.

---

### 25. ¿Qué organiza el sistema de archivos?
A) Solo la pantalla  
B) Los archivos en directorios y carpetas  
C) Únicamente la CPU  
D) Solo la red  

**Respuesta:** B  
**Solución:** El sistema de archivos permite crear, eliminar y organizar archivos y carpetas.

---

### 26. ¿Cuál de las siguientes es una estructura de sistema operativo mencionada en clase?
A) En capas  
B) En colores  
C) Circular  
D) Abierta fija  

**Respuesta:** A  
**Solución:** Una de las estructuras mencionadas es la estructura en capas.

---

### 27. ¿Qué sistema operativo se caracteriza por ser de código abierto?
A) Windows  
B) Linux  
C) OS/2  
D) MS-DOS  

**Respuesta:** B  
**Solución:** Linux se caracteriza por ser de código abierto.

---

### 28. ¿Cuál es una característica común de Windows?
A) Código abierto total  
B) Muy usado en servidores científicos únicamente  
C) Interfaz gráfica amigable  
D) Uso exclusivo en máquinas virtuales  

**Respuesta:** C  
**Solución:** Windows destaca por su interfaz gráfica amigable y su uso extendido en PCs personales.

---

### 29. ¿Qué fue OS/2?
A) Un navegador web  
B) Un sistema operativo desarrollado por IBM  
C) Un disco magnético  
D) Un lenguaje de programación  

**Respuesta:** B  
**Solución:** OS/2 fue un sistema operativo desarrollado por IBM y usado en entornos empresariales.

---

### 30. ¿Qué servicios brinda el sistema operativo?
A) Solo decoración visual  
B) Ejecución de programas, manejo de archivos, detección de errores y seguridad  
C) Solo impresión de documentos  
D) Solo acceso a internet  

**Respuesta:** B  
**Solución:** El sistema operativo ofrece varios servicios esenciales para que los programas funcionen correctamente.
