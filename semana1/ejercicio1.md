# Banco de preguntas – Semana 1: Introducción a los Sistemas Operativos

## Preguntas de opción múltiple

### 1. ¿Qué son los recursos en una computadora?
A) Solo los programas instalados  
B) Los elementos que los programas necesitan para funcionar  
C) Únicamente la memoria RAM  
D) Solo los dispositivos de entrada  

**Respuesta:** B  
**Solución:** Los recursos son los elementos que los programas necesitan para funcionar, como CPU, RAM, disco y dispositivos de entrada/salida.

---

### 2. ¿Cuál de los siguientes es un recurso?
A) Chrome  
B) Word  
C) CPU  
D) Un documento  

**Respuesta:** C  
**Solución:** La CPU es un recurso del sistema que los procesos necesitan para ejecutarse.

---

### 3. ¿Qué es un proceso?
A) Un programa guardado en el disco  
B) Un programa en ejecución  
C) Un tipo de hardware  
D) Un dispositivo de entrada  

**Respuesta:** B  
**Solución:** Un proceso es un programa que ya se está ejecutando.

---

### 4. ¿Qué diferencia hay entre programa y proceso?
A) No hay diferencia  
B) Programa es hardware y proceso es software  
C) Programa es un conjunto de instrucciones guardadas y proceso es ese programa en ejecución  
D) Programa usa recursos y proceso no  

**Respuesta:** C  
**Solución:** El programa está guardado; el proceso es ese programa funcionando en ese momento.

---

### 5. ¿Cuál es la función principal del sistema operativo?
A) Fabricar el hardware  
B) Administrar los recursos de la computadora  
C) Reemplazar a los programas  
D) Apagar la computadora automáticamente  

**Respuesta:** B  
**Solución:** El sistema operativo administra y organiza el uso de los recursos.

---

### 6. El sistema operativo se puede entender como:
A) Un periférico  
B) Un administrador  
C) Un proceso cualquiera  
D) Un tipo de memoria  

**Respuesta:** B  
**Solución:** Se compara con un administrador porque reparte y controla recursos.

---

### 7. ¿Qué relación correcta existe entre proceso, recurso y sistema operativo?
A) El recurso controla al sistema operativo  
B) El sistema operativo solicita procesos  
C) El proceso solicita recursos y el sistema operativo decide cómo repartirlos  
D) El hardware pide permisos al proceso  

**Respuesta:** C  
**Solución:** Esa es la relación central del tema: el proceso necesita recursos y el sistema operativo decide cómo asignarlos.

---

### 8. ¿Cuál de estos NO es un recurso mencionado?
A) Teclado  
B) Mouse  
C) Monitor  
D) Carpeta de tareas  

**Respuesta:** D  
**Solución:** Una carpeta de tareas no fue mencionada como recurso del sistema.

---

### 9. ¿Cuál de los siguientes pertenece al hardware?
A) Memoria RAM  
B) Sistema operativo  
C) Proceso  
D) Programa en ejecución  

**Respuesta:** A  
**Solución:** La memoria RAM es una parte física de la computadora, por eso pertenece al hardware.

---

### 10. ¿Qué hace el sistema operativo cuando varios procesos quieren usar la CPU?
A) Ignora a todos  
B) Reparte y organiza su uso  
C) Elimina los procesos  
D) Convierte la CPU en memoria  

**Respuesta:** B  
**Solución:** El sistema operativo decide cómo se comparte la CPU entre varios procesos.

---

### 11. ¿Cuál de estas opciones describe mejor al sistema operativo?
A) Es hardware  
B) Es un dispositivo de salida  
C) Es software que administra hardware y recursos  
D) Es una aplicación de oficina  

**Respuesta:** C  
**Solución:** El sistema operativo es software encargado de administrar el hardware y los recursos.

---

### 12. Cuando Chrome, Word y Spotify están abiertos al mismo tiempo, cada uno es:
A) Un recurso  
B) Un hardware  
C) Un proceso  
D) Un dispositivo de entrada  

**Respuesta:** C  
**Solución:** Al estar ejecutándose, cada uno se considera un proceso.

---

### 13. ¿Qué recurso se encarga del procesamiento?
A) Disco  
B) Monitor  
C) CPU  
D) Impresora  

**Respuesta:** C  
**Solución:** La CPU realiza el procesamiento de instrucciones.

---

### 14. ¿Cuál de los siguientes es un dispositivo de entrada?
A) Pantalla  
B) Impresora  
C) Teclado  
D) Disco duro  

**Respuesta:** C  
**Solución:** El teclado permite ingresar información al sistema.

---

### 15. ¿Cuál de los siguientes es un dispositivo de salida?
A) Mouse  
B) Teclado  
C) Pantalla  
D) Micrófono  

**Respuesta:** C  
**Solución:** La pantalla muestra información al usuario, por eso es de salida.

---

### 16. El sistema operativo decide:
A) Solo si da o no da recursos  
B) Cuándo, cuánto, por cuánto tiempo y en qué orden asigna recursos  
C) Solo qué programas borrar  
D) Solo qué color tendrá la pantalla  

**Respuesta:** B  
**Solución:** El sistema operativo no solo entrega recursos, también organiza su uso en tiempo, cantidad y orden.

---

### 17. ¿Qué afirmación es correcta?
A) El sistema operativo es un recurso  
B) El proceso administra al sistema operativo  
C) Los procesos necesitan recursos para funcionar  
D) El hardware solicita CPU al proceso  

**Respuesta:** C  
**Solución:** Los procesos requieren CPU, memoria, disco y otros recursos para poder ejecutarse.

---

### 18. ¿Cuál de estas opciones describe mejor a un programa?
A) Conjunto de instrucciones guardadas  
B) Programa ya ejecutándose  
C) Parte física de la computadora  
D) Dispositivo de salida  

**Respuesta:** A  
**Solución:** Un programa es un conjunto de instrucciones almacenadas.

---

### 19. ¿Por qué el sistema operativo es importante?
A) Porque evita que el hardware exista  
B) Porque permite organizar el uso de recursos y ejecutar programas correctamente  
C) Porque reemplaza la memoria RAM  
D) Porque convierte procesos en hardware  

**Respuesta:** B  
**Solución:** Su importancia está en que organiza el sistema para que los programas funcionen sin desorden.

---

### 20. Si un proceso necesita memoria, disco y CPU, ¿quién decide cómo se le asignan?
A) El teclado  
B) La impresora  
C) El sistema operativo  
D) El monitor  

**Respuesta:** C  
**Solución:** El sistema operativo controla la asignación y uso de esos recursos.
